# 🌸 She Can Foundation - Full Stack Development Internship Task

A beautiful, responsive contact form website for She Can Foundation built as part of the Full Stack Development internship application.

## ✨ Features

✅ **Contact Form** - Name, Email, and Message fields  
✅ **Form Validation** - Ensures all fields are filled and email format is correct  
✅ **Success Message** - Beautiful confirmation after form submission  
✅ **Responsive Design** - Works perfectly on all devices (mobile, tablet, desktop)  
✅ **Smooth Animations** - Engaging entrance effects and hover interactions  
✅ **Navigation Menu** - Sticky navbar with smooth scrolling  
✅ **About Section** - Information about She Can Foundation with images  
✅ **Professional Footer** - Clean and organized footer  
✅ **JavaScript Features** - Form handling, validation, and user feedback

## 🚀 How to Use

### Option 1: Open Locally
1. Download the `index.html` file
2. Double-click to open in your web browser
3. Fill in the form and test the functionality

### Option 2: View Online
1. Visit the live link above
2. Fill in the form with:
   - Your full name
   - A valid email address
   - Your message
3. Click the "Submit" button
4. See the success confirmation message

## 📋 Form Fields

| Field | Type | Required | Validation |
|-------|------|----------|-----------|
| Full Name | Text Input | Yes | Non-empty |
| Email | Email Input | Yes | Valid email format |
| Message | Text Area | Yes | Non-empty, min 10 chars |

## 🔧 Features Breakdown

### Form Validation
- Checks if all fields are filled before submission
- Validates email format using regex
- Shows user-friendly error messages

### Success Message
- Appears after successful form submission
- Animated entrance with scaling effect
- Automatically resets after 3 seconds for new submissions

### Responsive Design
- Mobile-first approach
- Breakpoints optimized for all devices
- Touch-friendly button sizes

### Animations
- Page load animations with staggered delays
- Hover effects on interactive elements
- Smooth transitions throughout
- Loading spinner on submit button
